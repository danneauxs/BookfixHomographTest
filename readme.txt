This program runs under Linux, Windows WSL, and Windows Native.
(Don't forget to make the files executable.)


For Linux install simply create a folder, move all the files there.
Make sure you have Python 3.12 installed.  3.13 and 3.14 will cause errors.
In Terminal runL
python3.12 -m venv .venv (creates python environment)
install_linux.sh
Then run run_linux.sh

For Windows WSL, use the same procedure as Linux install.

For Windows native, make a folder and copy all the files there.
Make sure you have Python 3.12 installed.  3.13 and 3.14 will cause errors.
py -3.12 -m venv venv (creates python environment)
install_windows.bat
run_windows.bat
